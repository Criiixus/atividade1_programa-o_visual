Este projeto foi desenvolvido como atividade da faculdade para praticar os conceitos básicos da linguagem C# utilizando Windows Forms no Microsoft Visual Studio.

A aplicação permite que o usuário digite seu nome em um campo de texto e, ao clicar em um botão, o sistema exibe uma mensagem de saudação personalizada utilizando MessageBox.

O objetivo da atividade é exercitar a criação de interfaces gráficas, manipulação de eventos, uso de controles visuais (Label, TextBox e Button) e aplicação de boas práticas de programação e nomenclatura em C#.
